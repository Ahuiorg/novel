# Change: 创建仙逆编年史沉浸式 Web 页面

## Why

《仙逆》小说拥有丰富的时间线和修炼体系，目前仅以 Markdown 文档形式存在。需要一个炫酷的、沉浸式的 Web 页面来展示王林 2288 年的修仙历程，让读者能够以交互式的方式体验这部经典修真小说的编年史。

## What Changes

### 新增页面
- **首页（时间线总览）**：沉浸式星空背景 + 垂直时间线 + 滚动触发动画
- **详情页（纪元详情）**：单个时间段的完整故事 + 收集品展示
- **等级体系页**：十步修真境界可视化展示
- **图鉴页**：神通/法宝/灵兽收集品汇总

### 核心特效
- Three.js 星空粒子背景
- 灵气流动效果
- 境界突破全屏动画
- 破碎虚空页面转场
- 鼠标跟随灵光效果

### 功能特性
- 搜索/快速跳转（按时间、地点、关键词）
- 阅读进度保存（localStorage）
- 环境音效系统（可开关）
- 暗夜星辰视觉主题

### 读者体验增强
- **名场面高亮**：每个纪元标记 1-2 个高光时刻
- **经典台词展示**：突出显示令人印象深刻的语录
- **分享卡片生成**：生成精美的分享图片
- **随机精彩**：首页"随机一段名场面"入口
- **人物标签**：详情页高亮重要人物名称
- **个人收藏**：收藏喜欢的纪元/神通/名场面

## Impact

- **新增能力**：
  - `homepage` - 首页时间线展示
  - `era-detail` - 纪元详情页
  - `level-system` - 等级体系页
  - `collection` - 图鉴收集页
  - `core-effects` - 核心特效系统
  - `reader-experience` - 读者体验增强（名场面、分享、收藏）

- **技术栈**：
  - Vue 3 + TypeScript + Vite
  - Three.js（3D 粒子特效）
  - GSAP（高级动画）
  - Naive UI（组件库）
  - Pinia（状态管理）
  - Howler.js（音频）
  - Vitest + Vue Test Utils（单元测试）
  - Playwright（E2E 测试）

- **数据来源**：`故事情节.md` 转换为结构化 JSON
