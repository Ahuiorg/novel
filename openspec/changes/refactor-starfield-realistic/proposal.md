# Proposal: refactor-starfield-realistic

## Summary

重构 StarField 组件的 Three.js 实现，保留 3D 旋转效果，但改进星星的分布、渲染和闪烁方式，实现更真实的仰望星空感觉。

## Motivation

当前星空效果存在以下问题：
1. **星星会消失** - 圆柱形分布 + y轴旋转导致视野内星星数量不稳定
2. **闪烁不自然** - 周期性的呼吸效果太机械
3. **缺乏层次** - 所有星星看起来差不多
4. **过度特效** - 星芒和流星效果太花哨

## Proposed Solution

保留 Three.js 3D 渲染，改进以下方面：

### 1. 星星分布
- **包围球** - 把相机放在巨大星空球体的中心，星星在四周
- **始终可见** - 无论怎么旋转，视野内都有星星
- **密度均匀** - 球面均匀分布

### 2. 视觉层次
- **星等系统** - 模拟真实星等（1-6等），大部分星暗，少数星亮
- **颜色差异** - 主要白色，少量暖色（橙红）和冷色（蓝白）
- **大小差异** - 亮星大，暗星小

### 3. 自然闪烁
- **随机闪烁** - 用 Perlin Noise 或随机算法，不是正弦波
- **部分闪烁** - 只有部分星星闪烁，不是全部
- **不规则** - 每颗星闪烁节奏不同

### 4. 旋转方式
- **缓慢旋转** - 整体缓慢旋转，有"星空流转"的感觉
- **平滑** - 旋转速度恒定，无跳跃

### 5. 简化特效
- **移除星芒** - 去掉带十字光芒的大星星
- **可选流星** - 流星效果简化或移除

## Technical Approach

```
Camera (z=0) 在球心
        ↓
   ★  ★  ★  ★
 ★    ★    ★    ★
★  ★  [相机]  ★  ★   ← 星星在球面上
 ★    ★    ★    ★
   ★  ★  ★  ★
```

- 相机在原点，星星分布在半径 100-500 的球面上
- 球体缓慢旋转，相机不动
- 使用 `THREE.Points` 渲染数千颗星星
- 自定义 Shader 实现闪烁效果

## Impact

- **文件变更**: `StarField.vue` 重构
- **依赖**: 保留 Three.js
- **性能**: 优化后应该更好

## Success Criteria

1. 视野内始终有满天繁星
2. 星空缓慢旋转，有"星辰流转"的感觉
3. 星星自然闪烁，像真实星空
4. 有明暗层次，不单调
