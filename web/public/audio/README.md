本目录存放网站所需的音效文件。

## 目录结构

```
audio/

    ├── click.mp3          # 点击音效
```

## 音效规格建议

### 环境音乐
- 格式：MP3 (192kbps+)
- 时长：30秒-2分钟（循环友好）
- 特点：可无缝循环，氛围感强

### 交互音效
- 格式：MP3 (128kbps)
- 时长：0.1-1秒
- 特点：清脆、短促

### 突破音效
- 格式：MP3 (192kbps)
- 时长：1-3秒
- 特点：有层次感，从低到高

## 推荐音源

1. **Freesound.org** - 免费音效库
2. **Pixabay** - 免版税音效
3. **Zapsplat** - 游戏音效
4. **Epidemic Sound** - 专业音效（需订阅）

## 注意事项

- 确保所有音效文件的版权许可允许商用
- 音量需统一标准化（-14 LUFS 推荐）
- 建议压缩后单个文件不超过 500KB
