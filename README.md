# 仙逆编年史 (Xianni Chronicle)

> 顺为凡，逆则仙，只在心中一念间

一个基于耳根著作《仙逆》的互动式编年史 Web 应用，带你重温王林从凡人少年到踏天境强者的传奇修仙之路。

## 项目简介

本项目是一个沉浸式的《仙逆》故事时间线浏览器，包含：

- 📅 **时间线浏览** - 从仙逆元年到仙逆2288年，完整呈现王林的修仙历程
- 🎭 **人物图鉴** - 收录重要角色信息
- ⚔️ **境界系统** - 详细展示修炼等级体系（炼气→筑基→结丹→元婴→...→踏天境）
- 📚 **典藏收集** - 神通、法宝、灵兽等分类收录
- ⭐ **收藏功能** - 标记你喜爱的内容
- 🌌 **视觉特效** - 星空背景、灵气粒子等沉浸式效果

## 技术栈

- **框架**: Vue 3 + TypeScript
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **UI 组件**: Naive UI
- **动画**: GSAP + Three.js
- **测试**: Vitest + Playwright

## 快速开始

```bash
# 进入 web 目录
cd web

# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 运行测试
pnpm test
```

## 时代划分

故事共分为 16 个重要时期：

1. **出生时期** (仙逆元年-15年) - 凡人少年
2. **入门时期** (15-101年) - 恒岳派入门
3. **修魔海时期** (102-410年) - 古神传承
4. **复仇时期** (411-530年) - 化神归乡
5. **仙遗之地时期** (530-600年) - 婴变突破
6. **天运星时期** (600-800年) - 妖灵历练
7. **罗天星域时期** (800-1050年) - 问鼎修为
8. **雷之仙界时期** (1050-1120年) - 阳实突破
9. **猎杀时期** (1170-1252年) - 窥涅净涅
10. **仙府大战时期** (1252-1260年) - 朱雀觉醒
11. **云海时期** (1300-1415年) - 碎涅修为
12. **界内外战争时期** (1416-1455年) - 空涅之战
13. **古之墓地时期** (1454-1530年) - 空灵境界
14. **封界大战时期** (1530-1570年) - 天道之主
15. **仙罡大陆时期** (1571-1987年) - 踏天之路
16. **终章** (1988-2288年) - 踏天成道
